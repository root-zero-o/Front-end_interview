# 식별자 네이밍 규칙

- 식별자 : 어떤 값을 구별해서 식별해낼 수 있는 고유한 이름(변수명)

- 네이밍 규칙

    - 식별자는 특수문자를 제외한 문자, 숫자, 언더스코어(```_```), 달러기호(```$```)를 포함할 수 있음
    - 단, 식별자는 숫자로 시작할 수 없음! 숫자를 제외한 문자, 언더스코어(```_```), 달러기호(```$```)로 시작해야 함
    - 예약어는 식별자로 사용할 수 없음
- 자바스크립트 예약어
